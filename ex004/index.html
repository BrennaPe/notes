Flex box

No Flexbox, temos dois eixos principais para alinhar os itens:

  • Eixo principal (main axis): é o eixo principal de alinhamento dos itens e depende da direção do flex-direction.
  • Eixo cruzado (cross axis): é o eixo perpendicular ao eixo principal e se ajusta de acordo com a direção definida no flex-direction.
  • Obs: Os itens vão sempre seguir e são sempre colocados no sentido do main axis.

É fundamental identificar corretamente quem é o flex-container e quem são os flex-items, pois o display: flex é aplicado no contêiner pai (a tag que aninha os itens se serem alinhados). Essa configuração transforma o contêiner em um flexbox e permite que os elementos filhos (flex-items) sejam posicionados e dimensionados de acordo com as propriedades flexíveis

Direções e eixos flex box CSS

Flex direction row

The flex container's main-axis is defined to be the same as the text direction. The main-start and main-end points are the same as the content direction.

A propriedade flex-direction em Flexbox define a direção principal (main axis) na qual os itens dentro de um container flex serão dispostos. Por padrão, essa direção é row, o que organiza os itens em linha, da esquerda para a direita.
