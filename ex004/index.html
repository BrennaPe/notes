É fundamental identificar corretamente quem é o flex-container e quem são os flex-items, pois o display: flex é aplicado no contêiner pai (a tag que aninha os itens se serem alinhados). Essa configuração transforma o contêiner em um flexbox e permite que os elementos filhos (flex-items) sejam posicionados e dimensionados de acordo com as propriedades flexíveis

A direção padrão (default) da propriedade flex é row (linha), o que alinha os itens do flex container da esquerda para a direita. se eu utilizar o flex-direction:row-reverse; (no container pau) continuaremos tendo os itens em linha porém teremos o modo reverso ou seja os itens estarão dispostos da direita para a esquerda. A mesma lógica serve para o column-reverse
