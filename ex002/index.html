Parágrafos e Quebras de Linha

Parágrafos são elementos de nível de bloco. O elemento HTML <p> representa um parágrafo. Em mídias visuais, parágrafos geralmente são representados como blocos de texto separados por linhas em branco e/ou por recuo/parágrafo na primeira linha. No entanto, em HTML, parágrafos podem agrupar qualquer conteúdo relacionado, como imagens ou campos de formulários.

Usar elementos <p> vazios para adicionar espaço entre parágrafos é problemático para pessoas que navegam com leitores de tela. Os leitores de tela podem anunciar a presença do parágrafo, mas não seu conteúdo — pois não há nada nele. Não importa se você quebra o texto apertando "Enter" no VSCode; no navegador, o texto sempre aparecerá em sequência. Para quebrar um parágrafo, é necessário usar a tag correta. Para isso, utiliza-se a tag <br> (Break Row, ou quebra de linha). Se você desejar um espaço extra, use propriedades CSS, como `margin`, para criar esse efeito
