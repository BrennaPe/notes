Operadores em JS

Operadores Relacionais

Comparison operators are used in logical statements to determine equality or difference between variables or values. Operadores relacionais (também chamados de operadores de comparação) em JavaScript são usados para comparar dois valores. O resultado de uma comparação sempre é um valor booleano: true ou false.

  • > Maior
  • < Menor
  • >= Menor ou igual
  • <=Menor ou igual
  • == Igual
  • === Igual em valor e tipo
  • != Diferente

Aqui estão os principais operadores relacionais em JavaScript:

Operador Descrição Exemplo (a = 5 , b = "5") Resultado
== Igual a == b true
=== Igual em valor e tipo a === b false
!= Diferente a != b false
!== Diferente em valor e tipo a !== b true
> Maior que a > 3 true
< Menor que a < 3 false
>= Maior ou igual a a >= 8 false
<= Menor ou igual a a <= 8 true

Em operadores relacionais as comparações de tamanho são resolvidas antes das comparações de igualdade.

Por exemplo:
5 > 3 == true
Primeiro: 5 > 3 → true
Depois: true == true → true

Dica importante:Use sempre === e !== em vez de == e !=, para evitar surpresas com a coerção de tipo automática do JavaScript. Coerção de tipo automática (ou type coercion) em JavaScript é quando o próprio JavaScript converte automaticamente um tipo de dado em outro para tentar resolver uma operação.

Um Exemplo classico de coerção é 5 == "5" resultando em TRUE. Aqui temos um número (5) sendo comparado com uma string ("5"). O JavaScript converte automaticamente a string para número antes de comparar. Por isso dá true.

Isso é coerção de tipo: ele força uma conversão pra tentar fazer a comparação funcionar.
