Important observation before exercícios

A tag <code> serve para indicar que um determinado trecho do texto é código — como comandos, variáveis, operadores, funções, etc.Ela é usada para dar significado semântico ao conteúdo (ou seja, dizer ao navegador, leitores de tela e buscadores que aquele texto é código), além de aplicar uma fonte monoespaçada (tipo de letra onde cada caractere ocupa o mesmo espaço, como em editores de código).

Exercícios op aritméticos:

let n = 3
n = n + 4 (n deixa de valer 3 e passa a valer 7 isso é auto atribuição)
n = n - 5 (o n que estava valendo 7 deixa de valer 7 e passa a valer 2 ... também temos a short version de auto atribuição que é da seguinte forma)
n*=4 (o n que tava valendo 2 passa a valer 8)
n/=2 (n passa a valer 4)
let a = 5+3 (result 8)
let b = a%5 (result 3)
let c = 5*b**2 (result 45)
let d = 10-a/2 (result 6)
let e = 6*2/d (result 2)
let f = b%e+4/e (result 3)

Operadores de incremento:

let x = 5
x = x + 1 (result 6)
x+=1 (result 6)
x++ (result 6)
Esses três exemplos são uma demonstração de uma mesma conta porém de três formas diferentes de faze-las
let x = 5
x = x-1 (result 4)
x-=1 (result 4)
x-- (result 4)

“O usuário tem permissão se for ADMIN ou MODERADOR.” ➡ Como ficaria a condição em JS?
user === "ADMIN" || user === "MODERADOR"

“Para entrar no evento, a pessoa precisa ter mais de 18 anos e ter ingresso.”
idade >= 18 && ingresso === "sim"
Dica: sempre use === ao invés de ==, para garantir que o tipo também seja igual.

“Verifique se o aluno passou. A nota precisa ser maior ou igual a 6.”
