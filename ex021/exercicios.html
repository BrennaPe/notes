Important observation before exercícios
A tag <pre> (de preformatted text) serve para manter o formato original do texto dentro dela — incluindo espaços, quebras de linha e identação — exatamente como está escrito no HTML. A tag <code> serve para indicar que um determinado trecho do texto é código — como comandos, variáveis, operadores, funções, etc.Ela é usada para dar significado semântico ao conteúdo (ou seja, dizer ao navegador, leitores de tela e buscadores que aquele texto é código), além de aplicar uma fonte monoespaçada (tipo de letra onde cada caractere ocupa o mesmo espaço, como em editores de código).
